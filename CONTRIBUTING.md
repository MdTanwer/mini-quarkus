# Contributing to Mini-Quarkus

Thank you for your interest in contributing! Mini-Quarkus is a learning project that demonstrates the core concepts of Quarkus.

## Development Setup

1. **Prerequisites**
   - Java 17+
   - Maven 3.8+

2. **Build**
   ```bash
   ./mvnw clean install
   ```

3. **Run Sample App**
   ```bash
   ./scripts/run-main-app.sh
   ```

## Contribution Guidelines

### Areas for Contribution
- **Core Features**: Gizmo bytecode generation, configuration system
- **Extensions**: Additional REST endpoints, validation, security
- **Documentation**: Tutorials, examples, API docs
- **Tests**: More integration tests, performance benchmarks

### Submitting Changes
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests if applicable
5. Submit a pull request

## Learning Resources

This project is designed to help understand:
- Build-time processing vs runtime reflection
- CDI container implementation
- HTTP server architecture
- Maven multi-module projects

## Questions?

Feel free to open an issue for questions or discussion!
